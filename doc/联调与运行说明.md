# 联调与运行说明

## 1. 后端启动
1. 创建数据库并执行 `doc/sql/init.sql`
2. 修改 `backend/src/main/resources/application.yml` 数据库账号密码
3. 在 `backend` 目录启动 Spring Boot

## 2. 用户端启动
```bash
cd library-user
npm install
npm run dev
```
默认地址：`http://localhost:5173`

## 3. 管理端启动
```bash
cd library-admin
npm install
npm run dev
```
默认地址：`http://localhost:5174`

## 4. 演示账号
- 管理员：`admin / 123456`
- 普通用户：`reader / 123456`

## 5. 当前实现范围
- 后端：用户认证、图书/分类/用户/轮播图管理、阅读记录、仪表盘统计
- 用户端：登录注册、首页推荐、图书列表/详情、阅读页、个人中心
- 管理端：登录、仪表盘、书籍管理、分类管理、用户管理、系统设置
