# 后端接口清单（Phase 2）

统一返回：`{ code, message, data }`

## 用户端
- `POST /api/user/register` 注册
- `POST /api/user/login` 登录
- `POST /api/user/mp/login` 小程序微信登录（code/unionId）
- `GET /api/user/info` 当前登录用户信息
- `POST /api/user/logout` 退出
- `GET /api/book/list` 书籍分页
- `GET /api/book/detail/{id}` 书籍详情
- `GET /api/book/read/{id}` 阅读入口（需登录）
- `POST /api/book/favorite/{bookId}` 收藏
- `GET /api/book/favorite/list` 收藏列表
- `POST /api/book/history` 保存阅读进度
- `GET /api/book/history/list` 阅读历史
- `GET /api/category/tree` 分类树
- `GET /api/carousel/list` 轮播图

## 管理端
- `POST /api/admin/login` 管理员登录
- `GET /api/admin/dashboard` 仪表盘统计
- `GET /api/admin/book/list` 图书分页
- `POST /api/admin/book` 新增图书
- `PUT /api/admin/book/{id}` 编辑图书
- `PUT /api/admin/book/{id}/status?status=1` 上下架图书
- `DELETE /api/admin/book/{id}` 删除图书
- `GET /api/admin/category/tree` 分类树
- `POST /api/admin/category` 新增分类
- `PUT /api/admin/category/{id}` 编辑分类
- `DELETE /api/admin/category/{id}` 删除分类
- `GET /api/admin/user/list` 用户分页
- `PUT /api/admin/user/{id}/status` 更新用户状态
- `GET /api/admin/system/carousel/list` 轮播图管理列表
- `POST /api/admin/system/carousel` 新增轮播图
- `PUT /api/admin/system/carousel/{id}` 编辑轮播图
- `DELETE /api/admin/system/carousel/{id}` 删除轮播图
